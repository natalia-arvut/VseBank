# VseBank · Вселенский Банк Изобилия

Духовный банковский симулятор — инструмент для практики изобилия на стыке метафизики и нейропластичности.

## Стек
- React 19 + TypeScript + Vite
- Tailwind CSS
- React Router
- EmailJS (отправка писем)

## Локальный запуск
```bash
npm install
npm run dev
```
Откроется на http://localhost:5173

## Сборка
```bash
npm run build
```
