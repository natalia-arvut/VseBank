# CLAUDE.md — Вселенский Банк Изобилия

## Контекст проекта

Духовный банковский симулятор — инструмент для практики изобилия на стыке метафизики и нейропластичности. Пользователь регистрируется, открывает виртуальный счёт, выбирает сумму, вводит реквизиты и «получает перевод от Вселенной». Это осознанная симуляция, цель которой — перепрограммировать убеждения о деньгах через эмоциональное проживание.

Исходные документы в папке:
- `лендинг.txt` — концепция, философия, описание механики симулятора
- `платформа.txt` — структура платформы по блокам (регистрация, кабинет, карты, реквизиты, переводы)

## Структура приложения

11 блоков согласно документу `платформа.txt`:
1. Регистрация (открытие счёта)
2. Пути самореализации (после регистрации)
3. Личный кабинет — приветствие
4. Главный экран кабинета
5. Ввод банковских реквизитов
6. Подпись и выход
7. Карты (5 штук)
8. Сроки поступления средств
9. Типы поступлений
10. От автора
11. Философский текст

## Требования к результату

- Веб-приложение: React + TypeScript + Tailwind CSS
- Без бэкенда — всё в localStorage и состоянии компонентов
- Никаких реальных платёжных систем и транзакций
- Дизайн: премиальный private banking, дорого и эксклюзивно
- Язык интерфейса: русский

## Правила работы с файлами

- Не удалять исходные документы (`лендинг.txt`, `платформа.txt`)
- Весь генерируемый код складывать в папку `src/`
- Итоговые отчёты и планы — в формате Markdown

## Формат ответа

- Сначала план, потом реализация после одобрения
- Короткий отчёт после каждого этапа
- Комментарии в коде — на русском
