## Что это за проект

**Вселенский Банк Изобилия** — веб-приложение-симулятор в жанре духовного банкинга. Инструмент для практики изобилия: пользователь проходит через реалистичный банковский интерфейс (регистрация, счёт, карты, переводы) как медитативную практику, направленную на перепрограммирование убеждений о деньгах.

Концептуальная основа — нейропластичность: мозг не отличает реальное переживание от воображаемого, если оно подкреплено сильной эмоцией. Симуляция банковского перевода, прожитая искренне, строит новые нейронные связи.

## Структура приложения (11 блоков)

### Блок 1 — Регистрация
Форма открытия счёта. Поля: имя, фамилия, дата рождения, пол, e-mail, пароль, страна. Чекбокс правил. Слоган: «С первой секунды ты — Банкир».

### Блок 2 — Пути самореализации
Экран после регистрации. Раздел «Разрешено для получения». Тема щедрости: 10% суммы — в фонд помощи детям с онкологией.

### Блок 3 — Приветствие в кабинете
Персональное приветствие. Призыв выбрать сумму согласно расчётам. Введение в «Счёт Всех Расчётов Банкиров».

### Блок 4 — Главный экран кабинета
Баланс (бесконечный / выбранный пользователем). Уведомление от администрации. Выписка № 07. Навигация: карты, реквизиты, переводы, налог.

### Блок 5 — Банковские реквизиты
Форма ввода: название банка, номер счёта, BIC/SWIFT, IBAN, спец. счёт 007, валюта. Активация карты перед выводом.

### Блок 6 — Подпись и выход
Экран подписания намерения. Кнопки: «Подпись», «Выход». Поздравление: «Завтра на вашем счету будет первый миллион».

### Блок 7 — Карты (5 штук)
Пять визуальных банковских карт с именем пользователя. Три уровня активации.

### Блок 8 — Сроки поступления
| Вариант | Срок |
|---|---|
| Быстрый | 1–3 дня |
| Стандартный | 5–7 дней |
| Расширенный | 10 дней |
| Максимальный | 21 день |

### Блок 9 — Типы поступлений
Два варианта: «Приход на личный баланс доверия» и «Приход напрямую — Автоперевод». Кнопки действий.

### Блок 10 — От автора
Мотивационный текст. Раздел с отзывами (только положительные).

### Блок 11 — Философский текст
Метафора об отце-миллиардере. Параллель с Небесным Отцом / Квантовым Полем как источником изобилия.

## План реализации

**Стек:** React + TypeScript + Tailwind CSS. Без бэкенда, без реальных платёжных систем. Данные — в localStorage.

**Этапы:**

1. **Дизайн-прототип** — в Claude Design, итерация по каждому экрану
2. **Регистрация и кабинет** — форма входа, личный кабинет с балансом
3. **Карты** — 5 визуальных карт с именем пользователя
4. **Симуляция перевода** — ввод реквизитов, выбор суммы и срока, подтверждение
5. **Финальные экраны** — подпись, поздравление, философские блоки
6. **Деплой** — GitHub + хостинг

**Дизайн-концепция:** премиальный private banking. Дорого, эксклюзивно, элегантно. Без массовости.
